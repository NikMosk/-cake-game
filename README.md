# 🍰 Cake Builder — Playable Ad

**Собери торт по слоям** — интерактивная рекламная игра с Drag & Drop механикой.

## 🎮 Механика
- Перетащи банку с ингредиентом на торт
- Слои добавляются в правильном порядке
- 6 слоёв: корж, варенье, крем, корж, сливки, вишня
- Таймер 45 секунд

## 📱 Поддерживаемые рекламные сети
- IronSource
- UnityAds
- Mintegral
- Moloco
- Liftoff

## ⚡ Технические характеристики
- **Вес:** < 2 МБ (с картинками)
- **Работает без интернета:** ✅
- **Адаптив:** ✅ (под любые экраны)
- **Трекинг событий:** ✅

## 🛠 Технологии
- HTML5 Canvas
- Чистый JavaScript
- Drag & Drop API
